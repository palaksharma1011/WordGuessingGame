## Description
A fun and interactive word-guessing game where players test their "Word Guesser: Test Your Vocabulary and Keep the Streak Alive!"
Word Guesser is my first backend project, combining creativity and functionality to deliver a fun and engaging game! Built using the random-words npm package, this game generates random words for you to guess, testing your vocabulary skills. It features a streak tracker that records how many correct guesses you can make in a row. Can you set a new high score? Dive in, break records, and enjoy this exciting word-guessing challenge! üéÆ‚ú®.
